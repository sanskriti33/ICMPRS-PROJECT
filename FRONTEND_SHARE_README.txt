Frontend Sharing Package

Use this if you want to send only the frontend app to teammates.

What to send:
1. The folder named `frontend-share`
or
2. The zip file named `frontend-share.zip` if present

How your teammates can open it:
1. Extract the zip if needed
2. Open the `frontend-share` folder
3. Double-click `Run-Frontend.bat`

Requirements on their laptop:
- Java installed
- MySQL available if they want live database connection

If MySQL is not available, the app may fall back to local file storage.
